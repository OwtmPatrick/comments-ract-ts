# Comments

A simple comments system

## Start

1. Clone [repo](https://github.com/OwtmPatrick/comments-react-ts)
2. Install packages

```
npm install
```

3. Start dev server

```
npm run dev
```

## Build

Run following command to build your source code:

```
npm run build
```
